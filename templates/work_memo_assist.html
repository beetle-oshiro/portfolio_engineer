Python / Flask / ChatGPT API / PostgreSQL

メモアシスト

ChatGPTAPI × Flask × PostgreSQLを用いて、メモの内容を整理してくれるシステムを制作します。

※無料版のFlaskを使用しているので、アクセスしてすぐはスリープ状態になっています。少し経ってから見れるようになります。数分触らないでいてもスリープ状態になります。

作品を開く

制作目的

  • ChatGPTにほとんどの作業を任せた場合、どれくらい簡単に、どのようなものを作成できるのか興味がありました。
  • 勉強中や仕事を始める際にとったメモの内容をAIに整理してもらうシステムがあると便利だと考え、ChatGPT APIと連携したメモアシストツールの制作を始めました。
  • このアプリでは、メモの内容を整理して保存し、カレンダー機能やワード検索機能でメモの内容を呼び出せるようにしていく予定です。

制作の流れ

  1. この制作物はChatGPTにコードは全て任せようと考えていたので、始めに「制作する目的」や「何ができるものか」、「どういうページ構成か」を伝えて制作していきました。
  2. 基本的にはChatGPTが出してくれるコードをVSCodeにコピペして動作確認し、エラーが出た時もChatGPTに相談しながら解決していきました。

操作方法

  1. テキストエリアにメモの内容を入力します。
  2. 入力後、「AIで整理して保存」ボタンを押します。 APIによる文章整理
  3. ページの下部にメモを整理した内容が表示され、データベースにも保存されます。
  4. 「保存済みメモの一覧を見る」を押すと、保存されたデータの一覧が表示されます。

今後の改善点

  • レスポンシブ対応とデザインを整える。
  • ワード検索機能やカレンダー機能を足します。

感じたことや反省点

  • コードを全てChatGPTに任せたので、私が作成する何倍も早くカタチになりました。
  • これまで何度か一緒に制作してきたことで、エラー等でつまづいた時の対処がより正確になってきました。
  • 協力することで早く作成できますが、この方法で制作物をつくることが正解なのか悩んでいます。
